# PickAShot
Save your web page as Image or Pdf

## Requirements
- Express JS
- MD5
- EJS Template Engine
- MongoDB
- Promise
- Puppeteer

## Test
Tested on 
- MacOS Sierra v10.12.5
- Node JS v8.4.0 (installed via nvm)
- MongoDB v3.4.7
